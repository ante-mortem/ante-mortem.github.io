Salut à tous ! :wave:

Il faut savoir que le serveur fonctionne avec à la fois des plugins (comme la majorité des serveurs MC) mais également avec des mods.
Les mods permettent l'ajout de blocs et de fonctionnalité qu'un plugin ne peux pas gérer. Un plugin est installé uniquement coté le serveur, alors qu'un mod est installé sur le serveur et également sur le client. Il existe aussi des mods qui ne sont que coté client comme la minimap ou optifine.
Afin de vous faciliter l'installation et la mise à jour des mods nous utilisons un lanceur personnalisé fournit via Tekkit.

  1. Télécharger "The Technic Launcher" : :link: http://www.technicpack.net/download
  2. Lancer l'exécutable que vous venez de récupérer (il est possible que vous n'ayez pas la bonne version de Java sur votre ordinateur, dans ce cas le launcher vous proposera d'aller le récupérer sur le site de Java)
  3. Entrer votre identifiant (pseudo ou adresse email) et votre mot de passe - Le launcher peux le sauvegarder afin de ne pas avoir à le saisir de nouveau.
  4. Aller dans l'onglet "MODPACKS"
  5. Dans le champ "Add Pack or Search" coller : http://api.technicpack.net/modpack/antemortem (ne donne rien dans un navigateur internet, c'est normal )
  6. Cliquer sur le bouton installer, le launcher récupéra alors le pack de mod et les données du jeu Minecraft
  7. Appuyer sur Jouer afin de lancer le jeu
  8. Enfin, une fois en jeu, aller dans Multiplayer et ajouter l'adresse : srv51.minecraft-mania.fr:27278 afin d'ajouter le serveur à la liste des serveurs.

Vous pouvez également aller dans Options du launcher afin de modifier certains réglage pratique comme :
- Ouvrir le launcher sur la page Modpacks automatiquement (cocher Start In Modpacks Tab)
- Ajuster la quantité de mémoire alloué à Java dans l'onglet Paramètres de Java et définir la version de java que vous souhaitez utiliser.

En cas de mise à jour du pack de mods du serveur, le launcher vous proposera de mettre à jour la version du pack de mods installé chez vous. La mise à jour vous sera proposé lorsque vous cliquerez sur le bouton Jouer.
Vous pouvez aller dans Options du modpack afin de réinstaller le pack ou ouvrir le dossier d'installation de ce dernier.
