Avec nos sessions arma j'ai eu la curiosité de tester l'éditeur avec des mods et c'est pas mal, les menus sont vite denses mais y a énormément de contenu. Du coup je testerai probablement la création d'une mission (le temps de tout setup ça risque de mettre un moment), probablement de défense d'une base (custom ou non je verrai) pour commencer, on verra par la suite.

Par contre avec les DLC et mods que j'utilise on va se mettre d'accord desuite :
- DLC Marksmen
- DLC Apex

Je préciserai si oui ou non il y a besoin d'un DLC pour une mission (bon à priori ce sera surtout pour les missions Tanoa via apex).

Coté mods j'utilise les mods suivants pour les missions customs :
CBA_A3
CUP Terrain-Core / Weapons / Units /  Vehicles / Terrain-Maps
HLC Core / M14 Pack / AK Pack / AR15 Pack
NIArms Core / FAL Rifles / MP5 SMGs / M1903 Springfield Rifles

Les joueurs d'arma moddé n'auront pas grand chose à DL ^^

Pour les missions d'ailleurs si vous avez une map préférée indiquez la sur ce topic (si vous ne comptez pas participer abstenez vous).

J'essaierai de vous laisser un équipement libre au début de chaque mission (peut être à thème, on verra selon les missions) via l'arsenal ou des caisses d'équipements.

Enfin, déja indiquez moi si ça vous tenterait et si oui sur quelle map vous voudriez jouer.

P.S : Le mod CUP Terrain-Maps contient les cartes suivantes :
Rahmadi
Sahrani
Southern Sahrani
United Sahrani
Porto
Utes
Chernarus
Desert
Takistan
Zargabad
Shapur
Proving Grounds
Bukovina
Bystrica
Takistan Mountains

Bon après avoir testé différents trucs sur l'éditeur et le mode Zeus, le mode Zeus reste plus pratique à utiliser même si moins complet.

Et comme dadau m'a aidé à tester les différentes options de création (spawns joueurs, véhicules, arsenal etc...) je vous proposerai une date sous peu pour une missions relativement simple : défense d'un lieu donné. La mission n'a pas de durée vraiment fixée du coup on verra si faire autre chose vous intéresse (ça me dérange pas de vous faire spawn des vagues entières d'ennemis mais ça risque de devenir répétitif).

Pour la map on verra selon les participants et leurs préférences.

Si vous voulez déjà vous préparer pour l'event vous pouvez déjà sauvegarder vos équipements via l'arsenal pour les récupérer directement après.

Bon après avoir fini de tester dans tous les sens le mode zeus, les mods et les différentes missions je peux vous proposer une date : vendredi 21, pour l'horaire je commencerai a setup la mission vers 20h je pense du coup on commencera plus ou moins quand vous voudrez une fois le setup terminé.

Pour information j'hébergerai la partie, je vous donnerai le mot de passe directement le soir même. Pas besoin de passer par Evolve ou ce genre de logiciels, normalement tout est ok de mon côté.

Vous avez plus qu'à vous préparer 2-3 stuff préconfigurés sur l'arsenal si vous voulez gagner du temps ^^

Pour rappel ce sera une mission de défense simple (du moins pour le début), si plus tard vous voulez changer vers une mission offensive ou autre ce sera possible (j'ai pas encore réussi à me débrouiller pour faire fonctionner les scripts de fin de mission du coup ça se fera à l'ancienne ^^).

Idem si certains d'entre vous veulent "masteriser" durant la soirée vous êtes libres de le faire, si vous comptez le faire je vous suggère le mode Ares qui ajoute quelques fonctionnalités utiles au Zeus (pas besoin du mod coté joueur) et je vous conseille également de vous y essayer au moins en solo avant, certaines commandes/config sont un peu tricky à utiliser et le placement d'objet ailleurs que sur le sol c'est tout un art !

Accessoirement j'ai tourné beaucoup sur Altis pour trouver un point défendable correctement à moins de 10 personnes, du coup vous défendrez la base militaire à l'ouest de l'airfield central d'Altis.

J'ai aussi testé différents ajouts des mods, y a des trucs que je ne vais clairement jamais vous filer (genre le M61 vulcain), d'autres qui sont daubés (les mitrailleuses lourds en tourelle qui se cassent la gueule si c'est pas parfaitement plat). Pour le reste si vous avez une préférence pour un véhicule, une tourelle, pièce d'artillerie etc... indiquez le moi (bon gardez en tête que si vous avez QUE des chars je vais pas envoyer que de l'infanterie hein ^^).

J'en profite aussi pour vous dire que si une IA se met soudainement à vous focus c'est probablement que j'en aurai pris le contrôle  :tatatatata: 

Comme je suis gentil j'essaierai de limiter l'artillerie côté IA puisque je serai obligé de la contrôler (ou de passer par des commandes) mais si c'est le cas je vous préviendrai avant de tirer, dans le cas des commandes, ou quand je tire pour l'artillerie contrôlée.
