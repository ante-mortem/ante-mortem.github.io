Qu'est-ce-que Starmade?​


[size=small]Star Made est un minecraft-like dont l’action prend place dans l’espace, Star Made est orienté massivement multijoueur avec des serveur starmade allant jusqu’à  plus de 999 slots. 


Explorer : explorer l’espace généré en temps réel autour de vous
Conception : conception de navire ou une station spatiale
Personnaliser : créer des armes personnalisées, accumuler des matériaux, fabriquer des boucliers et des stations d’accueil pour vos vaisseaux
Combiner : concevoir des éléments de groupe pour faire des modules encore plus puissants
Multijoueur : partager la réalisation de votre vaisseau, son commandement ainsi que son artillerie
Détruire : attaquer vos ennemis avec vos armes ou tout simplement infiltrer un vaisseau ennemi
Furtif : utilisation de brouillage radar et de module de camouflage pour des attaques surprise[/size]

Le jeu est 10€ sur steam(5€ en solde!^^') mais les dev on eu une bonne idée! Etant en Early Access le jeu est GRATUIT! C'est pas des bon dev ça?! Donc si vous le voulez pour l'instant aller sur steam et prendre la "demo"! Eh oui la démo est la version COMPLÈTE pour l'instant.

[font=Verdana, Helvetica, sans-serif]Site du développeur :  [/font][font=Verdana, Helvetica, sans-serif]http://star-made.org/[/font]

Quelque changement:

Amélioration de l'IA des tourelles!(J'aime ce jeux grâce au tourelles) J'ai aussi appris que maintenant les tourelles partages le bouclier(50%) du vaisseaux! Avant les tourelles se fessaient de suite degommer parce que même si elle était docker elle ne prenait pas le bouclier! De se faite maintenant il est possible de crée de petite tourelles! J'aime!

Un gros ajout qui se fessait tarder: Le mode créative... C’était chiant de se give en solo les blocks pour construire x) Le mod créative est activable en solo/mutli avec une commande ( http://www.starmade.fr/threads/sta...-turret-lag-jump-inhibitors-admin-tools.4837/ )

Et un nouveau ajout de blocks:
Le shipyards (Chantier naval) permet de charger des blueprints et de "construire" les vaisseaux Je pense que l'ajout de ses block est surtout pour une raison "Rp" car avant(Et encore maintenant Faut juste voir avec les serv s'ils autorisent encore cette manip Je le dit car il est bien sur possible de la rendre impossible dans les configs ) ils suffisaient d'avoir le blueprint dans l'inventaire et les ressources suffisant et le vaisseaux apparaissait. Il est possible aussi avec le Shipyard d'avoir un genre de "creatif" sur un serveur Une fois fini et les ressources suffisante on peut "construire" le vaisseaux Une autre fonction assez utile, il est possible de tester le vaisseaux: le shipyard "téléporte" dans un secteur virtuel où il est possible de tester notre navire Dans d'autre mise à jour des cibles pourront être mise dans se secteur enfin de tester l'armement (http://www.starmade.fr/threads/starmade-v0-194-shipyards.4793/ )
