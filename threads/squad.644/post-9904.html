Update 3.9

Ptite maj interessante aujourd'hui 130Mo~ J'ai fais la trad a partir de Gogole trad en ameliorant Corok saurais mieux nous dire ^^



Salut troufions,

Nous espérons que vous avez tous passez de bonnes fêtes, et maintenant que la majorité de l'équipe de développement est de retour au travail, nous aimerions deployer une nouvelle mise à jour, La 3.9 de la version Alpha actuelle de Squad. Contient un certain nombre d'améliorations utile dans Steam, des tas de mises à jour, des améliorations de performance du navigateur de serveurs et autres goodies.



Alpha 3.9 Changelog

- Optimisation des performances côté Client dans le système d'animation
- FIX du System de changement de classe qui ne fonctionné pas correctement
- Mise à jour des modèles de joueurs Milice avec de nouveaux composants, en les distinguant des autres factions
- Amélioration des performances sur l'ensemble des modèles de joueurs Milice
- Quiter une escouade remplace maintenant votre classe
- L'authentification améliorée avec Steam pour une meilleure stabilité
- Possibiliter de rejoindre ses amis via Steam
- Possibiliter de rejoindre des serveurs via des sites de votre navigateur Web
- Augmentation de connexion au serveur timeout, en diminuant la quantité de déconnexions accidentelles
- Augmentation de délai d'attente de navigateur de serveur
- Augmentation de délai d'attente pour rejoindre une Map pour permettre aux petite config de rejoindre sans être expulsé involontairement par le serveur pour temporisation
- Ajouté à Steam showplayerids
- Correction Bannissement par ID
- Amélioration des performances des effets d'explosion
- Ajout de la possibilité de réinitialiser les raccourcis clavier
- Appuyer sur le bouton " Par Defaut" permet maintenant de ne reinitialiser que le menu actif
- Les joueurs peuvent maintenant Annuler rejoindre le champ de tir dans le menu
- FIX Acces incorect au serveur pour les bannis
- Correction de quelques pannes de serveur

Aparement il y'aura d'autre maj ce mois ci ! c'est ca que c'est bon !


Original :


Hi Squaddies,

We hope that you all had a great New Years, and now that a majority of the development team is back at work, we’d like to put out a new update, version 3.9 of the current Alpha build of Squad. Contained are a number of Steam usability improvements, heaps of performance updates, Server browser improvements and other goodies.

Alpha 3.9 Changelog


Improved client performance related to optimizations in the anim system
Fixed kit allocations not working in many situations
Updated Militia player models with new components, distinguishing them further from other factions
Performance improvements to the Militia player models
Leaving a squad now replaces your kit
Improved Steam authentication for better stability
Allows for joining on Steam friends
Allows for joining servers via web browser sites
Increased server connection timeout, decreasing the amount of unintentional disconnects
Increased server browser timeout
Increased timeout for joining a map to allow people with low spec pc’s to load the game, without being kicked unintentionally by the server for timeout
Added SteamID to showplayerids
Fixed banning by ID issues
Improved performance of explosion effects
Added the ability to reset keybindings
Separated out the default button in each section of the settings menu, pressing default now only resets the current menu’s settings.
Players can now cancel joining the firing range in menu
Fixed incorrect access level for banning
Fixed a few server crashes


We’ve still got heaps in the chamber coming later this month, so please stay tuned to our webpage[joinsquad.com], forums[forums.joinsquad.com] and Steam community page for the latest.

Offworld Out.
