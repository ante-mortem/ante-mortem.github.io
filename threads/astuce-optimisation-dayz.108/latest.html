Je vous partagent ces quelques petites astuces pour optimiser Dayz

Je m'adresse à tous les possesseurs de carte Nvidia (les amd je ne sais pas si il existe une option équivalente dsl ;/ )



Allez dans le menu démarrer, puis dans la barre de recherche taper "nvidia",

cliquer sur le panneau de configuration de nvidia,

dans la panneau de configuration regarder la section paramètre 3D,

cliquer sur "Gérer les paramètres 3D", cliquer sur "Paramètres de programme",

sélectionner le programme que vous voulez modifier (ici on modifie Dayz ),

cliquer sur "Mode de gestion d'alimentation" sélectionner "Privilégier les performance maximale"

Enregistrer et maintenant vous pouvez jouer presque sans freeze ( enfin dépend du niveau de votre carte graph ^^)


Un petit bonus :
Pour supprimer l'animation de Bohemia avant le chargement de Dayz.

Allez dans les propriétés du jeu Dayz dans Steam, cliquer sur "Définir les options de lancement",
taper dans la barre "-nosplash" mettre un espace avant.

Un autre astuce qui permet de réduire un peu le temps de connexion à un serveur
Même manipulation que ci-dessus mais juste après le "-nosplash" taper "-noPause" rajouter un espace entre les deux.

PS: cette astuce marche pour tous les jeux

EDIT: un énorme merci à Edvard pour toutes ces astuces ^^ ( d'autres astuces seront bientôt éditer sur ce même poste )

Avec ces différents réglages et explications, vous devriez vous en sortir pour régler votre jeu au mieux.

A cause du moteur utilisé par Dayz SA, même un ordinateur récent et très performant est obligé de voir les options à la baisse. L'avantage cependant est que les configurations plus modestes elles peuvent bénéficier des ces options moins gourmandes et pouvoir jouer au jeu malgré la vétusté de l'installation.

Correction, l'option "-noPause" empêche seulement le jeu de se mettre en pause pendant un "Alt+Tab" par exemple.

Légende :
- GPU = Carte graphique
- CPU = Processeur
- Mes paramètres
Options en jeu :

-->Onglet : Video
---->Résolution = la résolution de votre écran 1920*1080
---->Overall Quality = Custom Custom
---->VSync = Disabled (sauf si vous avez un écran 120Hz) Disabled

-->Onglet : Quality
---->Objects = Very Low - Low pour CPU AMD (duo-quad cores) et Low - Normal pour CPU Intel (duo-quad cores) --- Normal - High pour CPU AMD (octo-core+) et High+ pour CPU Intel (octo-core+) Low
---->Terrain = Régler en fonction de votre GPU, consomme beaucoup de ressources GPU. Low
---->Clouds = Very Low ou Disabled Disabled
---->Shadows = Régler en fonction de votre GPU, consomme beaucoup de ressources GPU et modérément de ressources CPU. Low

-->Onglet : Textures
---->Video Memory = Auto Auto
---->Texture Detail = Régler en fonction de votre GPU, consomme beaucoup de ressources GPU. Normal
---->Texture Filtering = Régler en fonction de votre GPU, consomme peu de ressources GPU. High

-->Onglet : Rendering
---->Antialiasing = Régler en fonction de votre GPU, consomme beaucoup de ressources GPU. Low
---->Alpha to Coverage = All trees + grass (si l'Antialiasing est activé) All tress + grass
---->Edge Smoothing = Disabled Disabled
---->HDR Quality = Very Low Low
---->Ambient Occlusion = Disabled Disabled
---->Post Process Quality = Enabled Enabled
---->Bloom = Régler à votre convenance 0
---->Rotation Blur = Régler à votre convenance 0


Tests effectués sur la configuration suivante :
CPU : AMD Phenom II X4 965 Black Edition
MotherBorad : Asrock 970 Extrem4
GPU : MSI Nvidia Twin Frozor R7970 3GD
RAM : Corsair Vengeance 8GB
OS : Windows 7 Ultimate 64 bits

Avec ces réglages, j'obtiens un jeu joli et très fluide. 50FPS au plus bas.
Le facteur le plus décisif pour la "beauté" du jeu est l'option "Texture Detail" qui détermine le rendu de TOUTES les textures.

N.B : Je pourrais pousser les options plus haut, mais je ne souhaite pas avoir la moindre chute inopinée de FPS, un peu de qualité graphique sacrifiée pour de la jouabilité, c'est peu cher payé.
