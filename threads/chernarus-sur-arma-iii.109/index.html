Voici un petit tuto pour installer et jouer sur la map du Stand-Alone avec Arma III


Allez dans le dossier Arma 3 (C:\Program Files (x86)\Steam\SteamApps\common\Arma 3\)

Créez un dossier nommé "@ChernarusPlus" (sans les guillemets) et dans ce même dossier créez un autre dossier nommé "Addons"

Maintenant allez dans le dossier Dayz (C:\Program Files (x86)\Steam\SteamApps\common\DayZ\)

Copiez (ne déplacez pas) tous les fichiers qui se trouvent dans "Addons" sauf ceux ci-dessous;

characters_data.pbo
characters_data.pbo.dayz.bisign
characters_zombies.pbo
characters_vests.pbo
characters_tops.pbo.dayz.bisign
characters_tops.pbo

characters_proxies.pbo.dayz.bisign
characters_pants.pbo.dayz.bisign
characters_masks.pbo.dayz.bisign
characters_heads.pbo.dayz.bisign

characters_glasses.pbo

characters_proxies.pbo

modules.pbo.dayz.bisign

structures_ponds.pbo.dayz.bsign

ui.pbo
ui.pbo.dayz.bisign
uifonts.pbo
uifonts.pbo.dayz.bisign

- Ensuite collez tout ça dans le dossier "Addons" que vous avez créez dans "Arma 3\@ChernarusPlus"

Lancez Arma 3, ne prenez pas en compte les différents messages d'erreurs (faites "ok" ou "annuler tout")

Allez dans "Configurer", puis dans "Extensions"

Activez "@ChernarusPlus" et faite "OK", quand il vous dit "il faut redémarrer le jeu..." faites "OK" et non "REDEMARRER", quittez le jeu et relancé-le.

Il ne vous reste plus qu'à aller dans l'éditeur, choisir la map Chernarus et faire ce que bon vous semble...

NB : les fichiers de Dayz sont ceux correspondants à la dernière version installé via Steam (que ce soit la stable ou l'EXP, mais le contenue peut varié sur l'EXP, à vous de choisir...) . Pour avoir les nouveaux ajouts de chaque màj il faudra recopier les fichiers).


Voilà Enjoy !!!!

Edit : le tuto n'étant plus à jour je le laisse en l'état tout de même en attendant de retrouver une soluce qui marche
